WMS ENVIRONMENT MATRIX

Description

This web page provides a view into the WMS currently it is mostly aimedat giving operators acess to the databases so that they are able to see the production status of server.

Instructions

Use the links on the left for various breakdowns of the server Details.

When using the Search for Server option use '%' as a wildcard to match unknown characters. For example, 'u%mq%' will return all hostnames starting with 'u' and containing 'mq'.

Meaning of Fileds

Hostname

Generally the hostname of the system , but sometimes it is the first field of the fully qualified domain name instead.

OS

The Operating system installed on the server.
